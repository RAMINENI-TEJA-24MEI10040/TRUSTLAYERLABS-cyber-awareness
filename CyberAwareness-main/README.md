# CyberShield — AI Powered Cyber Awareness Platform

CyberShield is an advanced cyber awareness and Indian cyber law education platform built using React, TypeScript, Vite, Firebase, and AI-powered guidance systems.

---

# Features

- AI Scam Analyzer
- Context-Aware AI Assistant
- Cyber Awareness Modules
- Indian Cyber Law Education
- Real-Time Threat Alerts
- Firebase Authentication
- Admin Dashboard
- Incident Reporting Guidance
- Interactive Quiz System
- Hindi + English Language Support
- Mobile Responsive Design

---

# Tech Stack

## Frontend

- React
- TypeScript
- Vite
- TailwindCSS

## AI

- Groq API
- Lightweight RAG System

## Authentication

- Firebase Authentication

## Charts & UI

- Recharts
- Lucide React

---

# Installation

```bash
npm install
npm run dev
```

---

# Environment Variables

Create `.env`

```env
VITE_GROQ_API_KEY=

VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=
```
